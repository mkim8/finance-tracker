== README

This is the finance tracker app from the Complete Ruby on Rails Developer course.

- Build form
- Route -> search_stocks
- Controller ->action - stocks_controller.rb search

Stock Restrictions

- Users can add max 10 stocks
- Users cannot add a stock twice
Since this is a restriction on the User - Add it to the user model
